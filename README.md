# Family Photos App
![License:MIT](https://img.shields.io/badge/License-MIT-blue)

## Description
This app will take uploaded phtos and display them based on either date of upload or based on tags on the photo.   Users with granted access can edit tags on photos.  Comments are allowed on all photos, montitored by admin.
